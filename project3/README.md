## Обязательные функции

Реализованы следующие запросы:

#### `POST /links/shorten`

Создать короткую ссылку. Поддерживает параметры (опционально):

* expires_at - дата, до которой ссылка будет доступна
* custom_alias - возможность задать свой алиас, если он свободен

#### `GET /links/`

Открыть оригинальную страницу по короткому коду.

#### `DELETE /links/`

Удалить короткую ссылку. Доступно только авторизованным пользователям.

#### `PUT /links/`

Обновить ссылку по короткому коду. Доступно только авторизованным пользователям.

#### `GET /links//stats`

Получить информцию о короткой ссылке.

#### `GET /links/search?original_url={url}`

Найти активные короткие ссылки, которые ведут на `url`.

## Дополнительные функции:

Из дополнительных функций реализовано:

* Удаление неиспользуемых ссылок (будут удалены ссылки, которые не использовались в течение 5 дней)
* Отображение истории всех истекших ссылок с информацией о них по запросу `GET /links/deleted`
